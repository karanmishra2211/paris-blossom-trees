What the agent did

  1. Discovered the official Paris trees dataset and inspected its schema.
  2. Profiled species labels to find blossom-heavy candidates.
  3. Filtered Cerisier a fleurs rows and exported coordinates.
  4. Generated both interactive and static maps for class-ready use.

Why this is useful in class

The point is not just making a map. It is compressing the first hour of exploratory analysis into a short, inspectable workflow: data discovery, subset definition, verification, and visual output.

This is a concrete example of using AI agents for data exploration rather than only text generation.
