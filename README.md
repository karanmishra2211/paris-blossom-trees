# Paris Blossom Trees

Small GitHub Pages site for a classroom demo on AI-assisted data exploration.

It uses the official Paris open tree dataset, filters `Cerisier a fleurs`, and
publishes both a static presentation map and an interactive map.

Site entrypoint:

- `docs/index.html`

Supporting scripts:

- `paris_leaf_blossom_map.py`
- `paris_leaf_png.py`
